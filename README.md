## Commands

### `dvm version`

Show the current version of dvm.

### `dvm help {commandName}`

Show the help of dvm or specified command.

### `dvm install {version@channel}`

If nothing is specified, install the described Dart SDK of version and channel in `${currentDir}/.dvm/dvm_config.json`.

Install the specified Dart SDK of version and channel.

The installed Dart SDK is saved in `$HOME/.dvm/versions/{channel}/{version}`.

### `dvm use {version@channel}`

Use the specified version and channel of Dart SDK.
